## 🆘 TROUBLESHOOTING RAPIDE

### OCR ne marche pas bien
**Solution:**
- Vérifier qualité images (résolution >1000px)
- Tester avec Google Vision API (meilleur mais payant)
- Ajouter preprocessing images (contraste, rotation)

### Claude API trop cher
**Solution:**
- Utiliser Haiku pour classification simple
- Cacher résultats fréquents
- Batch requests quand possible

### Utilisateurs n'utilisent pas
**Solution:**
- Onboarding amélioré (vidéo + guide)
- Emails rappel hebdomadaires
- Feature manquante? Demander feedback

### Churn élevé
**Solution:**
- Exit survey pour comprendre pourquoi
- Améliorer onboarding
- Ajouter features demandées
- Support plus réactif
